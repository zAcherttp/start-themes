# Start Themes Showcase

This is a **personal showcase project** illustrating a powerful, hydration-safe, cross-tab synced theme hook built specifically for [TanStack Start](https://tanstack.com/start).

> **Note:** This is a personal endeavor and is **not** officially related to the TanStack ecosystem.

The beautiful documentation layout and the gorgeous color presets (Ocean, Aspen, Ruby, Catppuccin, etc.) used in this demo are gracefully provided by the phenomenal [Fumadocs](https://fumadocs.vercel.app/) library.

## Running the Showcase

Run the development server locally to preview the magic:

```bash
bun run dev
# or npm run dev
# or pnpm dev
```

Visit `http://localhost:3000` to see the seamless View Transition preset switcher in action, and click "useTheme Hook" to see the documentation.
