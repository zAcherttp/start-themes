Once you have copied the `use-themes.ts` hook into your project, you'll need to inject it into your root document so it can guarantee hydration safety and prevent FOUC (Flash of Unstyled Content).

## Update `__root.tsx`

The `useTheme` hook relies on a `ThemeState` object being available in your TanStack Router context. You'll need to use `createRootRouteWithContext` to define this.

Additionally, the `themeScript` string must be injected as an inline script inside your `<head>` tag.

import { Accordion, Accordions } from 'fumadocs-ui/components/accordion';

<Accordions>
<Accordion title="Why are these two things necessary?">

- **Router Context (`ThemeState`)**: TanStack Start apps render on the server. By fetching the theme cookie in `beforeLoad` and placing it in context, we inject the synchronized state directly into React's SSR stream. This completely eliminates React hydration mismatch errors.
- **Script Injection (`themeScript`)**: Server-Side Rendering (SSR) always emits the actual HTML markup *before* client-side Javascript runs. This inline script blocks the parser instantly just before css styles load to inject the proper Tailwind `.dark` and preset `.ocean`/`.shadcn` classes directly onto the `<html>` element based on cookies and OS settings. Without this, your users would see a "Flash of Unstyled Content" (a split-second bright flash) on every page refresh!

</Accordion>
</Accordions>

```tsx title="src/routes/__root.tsx"
import { createRootRouteWithContext, HeadContent, Outlet, Scripts } from '@tanstack/react-router';
import * as React from 'react';
import { getThemeServerFn, themeScript, type ThemeState } from '@/hooks/use-themes';
import appCss from '@/styles/app.css?url';

interface MyRouterContext {
  theme: ThemeState;
}

export const Route = createRootRouteWithContext<MyRouterContext>()({
  // 1. Fetch the user's persisted cookie using the server function
  beforeLoad: async () => ({ theme: await getThemeServerFn() }),
  head: () => ({
    meta: [
      { charSet: 'utf-8' },
      { name: 'viewport', content: 'width=device-width, initial-scale=1' },
    ],
    links: [{ rel: 'stylesheet', href: appCss }],
  }),
  component: RootComponent,
});

function RootComponent() {
  return (
    <RootDocument>
      <Outlet />
    </RootDocument>
  );
}

function RootDocument({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en" suppressHydrationWarning>
      <head>
        {/* 2. Block rendering and inject the CSS theme actively before paint */}
        {/* biome-ignore lint/security/noDangerouslySetInnerHtml: theming script */}
        <script dangerouslySetInnerHTML={{ __html: themeScript }} />
        <HeadContent />
      </head>
      <body>
        {children}
        <Scripts />
      </body>
    </html>
  );
}
```
