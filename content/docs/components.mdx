Once the hook is securely wired into your root router, you can consume it anywhere in your TanStack Start application to easily build mode toggles or color preset switches.

## Switching Modes (Dark/Light)

Use `modePreference` and `setMode` (or `nextMode`) to toggle the theme mode.

```tsx title="src/components/ModeToggle.tsx"
import { useTheme } from "@/hooks/use-themes";

export default function ModeToggle() {
  const { modePreference, nextMode } = useTheme();

  return (
    <button
      type="button"
      onClick={nextMode}
    >
      Mode: {modePreference}
    </button>
  );
}
```

## Switching Presets

Likewise, you can switch between pre-configured aesthetic themes like `shadcn`, `ocean`, `ruby`, and more!

```tsx title="src/components/PresetToggle.tsx"
import { useTheme } from "@/hooks/use-themes";

export default function PresetToggle() {
  const { presetPreference, nextPreset } = useTheme();

  return (
    <button
      type="button"
      onClick={nextPreset}
    >
      Preset: {presetPreference}
    </button>
  );
}
```

## How It Works

1. Calling `setMode` or `setPreset` effortlessly saves the new preference to the user's cookies via TanStack `createServerFn` actions.
2. A `BroadcastChannel` instantly alerts any other open tabs regarding the change, maintaining synchronized state without hydration mismatches.
3. The hook automatically freezes the DOM using `document.startViewTransition()`, safely syncing React's state inside the snapshot to provide beautiful cross-fading!
